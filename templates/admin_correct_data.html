Important: All three column names are case-sensitive and must appear at their designated positions. If your Excel file uses different names (e.g. pan card, aishe code, roll no), the system will automatically detect and rename them using fuzzy matching — but only if they are in the correct column positions (1, 10, 11). Files that do not follow this structure cannot be accepted by the university portal.
